# Course Champion

Course Champion is a responsive course listing website built using HTML, CSS, and Bootstrap.

## Features
- Fully responsive design
- Modern UI layout
- Course cards with hover effects
- Bootstrap grid system
- Clean and reusable components

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Font Awesome
- Remix Icons

## How to Run
1. Clone the repository
2. Open `index.html` in a browser
