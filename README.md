# Course Champion

Course Champion is a modern, responsive course listing website built using **HTML, CSS, Bootstrap, and JavaScript**.  
It features a clean UI, interactive components, and smooth animations designed to simulate a real-world education platform.

---

## Features

- Fully responsive layout (desktop, tablet, mobile)
- Modern indigo-based theme with reusable CSS variables
- Horizontal course slider with arrow navigation
- Smooth scroll-based animations
- Interactive hover effects on course cards
- Email subscription validation using JavaScript
- Clean and scalable component structure

---

## Technologies Used

- **HTML5**
- **CSS3**
- **Bootstrap 5**
- **JavaScript (ES6)**
- **Font Awesome**
- **Remix Icons**

---

## Key Implementations

- Custom horizontal slider built with JavaScript (`scrollBy`)
- Scroll snapping for smooth card alignment
- Intersection Observer for fade-in animations
- CSS variables for consistent theming
- Modular and maintainable styling approach
